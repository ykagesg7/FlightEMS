名前を暗記するんやない。「こいつは将来ヤバくなるシステム（雲）か？」「この裏にバケモノが隠れとらんか？」という『手配書』として雲を見れんヤツは、空の上で不意打ちを食らうんよ。

## 📚 詳細解説：雲の「一生」と「手配書」をハックせよ

学科試験で問われる「10種雲形」は、ただの暗記名やない。大気が生み出すエネルギーの履歴書たい。

### 1. 雲の変身：発生と成長のメカニズム

そもそも雲はどうやって生まれ、あんな巨大なバケモノに育つのか？
その引き金は、[大気の安定度（PPL-2-1-5）](/articles/PPL-2-1-5_AtmosphericStabilityBasics)で学んだ「空気の上昇」たい。

1.  **空気の上昇 ➡ 露点への到達（凝結の始まり）：**
    地面が温められたり、山肌に風がぶつかって空気が上昇すると、断熱膨張で冷えていく。気温が露点温度（結露する温度）に達した瞬間、見えない水蒸気が一斉に水滴に変わる。これが「雲の誕生（雲底高度）」たい。
2.  **潜熱エンジンの点火：**
    水蒸気が水滴に変わる時、前回の [水蒸気の物理と露点（PPL-2-1-4）](/articles/PPL-2-1-4_MoistureHumidityDewpoint) でやった**「潜熱（凝結熱）」**を大気中に放出する。
    もし空気が「条件付不安定」な状態やったら、この熱によって風船（空気の塊）は周囲より温かい状態をキープし、冷えにくい湿潤断熱減率（約1.5℃/1,000ft）で、自力でどこまでも上昇し始めるんよ！
3.  **バケモノ（積乱雲）への急成長：**
    こうして上昇気流がブーストされると、最初はカワイイ「積雲（Cu）」やったものが、数十分の間にモクモクと巨大化して「雄大積雲（TCu）」になり、最終的には成層圏の入り口まで突き抜ける巨大な「積乱雲（CB）」へとバケモノみたいに成長するんよ。

この「わずか数十分」という恐ろしい時間軸を知らんと、逃げ場を失って雷雲の中に南無さん特攻することになるんたい。

### 2. 10種雲形の仕分け：2つの「形」と3つの「高さ」

この大気のエネルギーの違いによって、雲は「形（積雲・層雲）」と「高さ（上・中・下）」の組み合わせで10種類に分類される。

*   **積雲系（モクモク・対流性）：** 下から上へ空気が吹き上がる（不安定な）時にできる。縦に成長し、**「乱気流（激しい揺れ）」**や**「にわか雨（しゅう雨）」**をもたらす。ヤンチャな暴れん坊たい。
*   **層雲系（ベターッ・層状）：** 空気が横に穏やかに広がる（安定な）時にできる。空をベターッと覆い、**「連続的な雨（シトシト雨）」**を降らせる。大人しいが、長時間視界（IFR）と着氷のストレスを強いてくる。

### 3. ひと目でわかる！10種雲形の定性対照表

学科試験で狙われる10種の名称と、その高さ、特徴、降水の性質をここで完璧にリンクさせるばい！

| 分類（高さ） | 雲形名（略号） | 俗称（イメージ） | 形状（特徴） | 降水・運航への影響（試験用） |
| :--- | :--- | :--- | :--- | :--- |
| **上層雲**<br/>（約20,000ft以上） | **巻雲 (Ci)** | すじ雲 | ハケで掃いたような白い繊維状の雲。氷の結晶。 | 降水なし。 |
| | **巻積雲 (Cc)** | うろこ雲 / さば雲 | 小さな白い塊が群れをなす。 | 降水なし。 |
| | **巻層雲 (Cs)** | うす雲 | 空一面をベールのように覆う。太陽に「かさ」を作る。 | 降水なし。 |
| **中層雲**<br/>（約6,500〜20,000ft） | **高積雲 (Ac)** | ひつじ雲 | 巻積雲より大きな白い塊。**「高」がつくが中層雲！** | 降水なし（たまに雨を降らせない尾流雲）。 |
| | **高層雲 (As)** | おぼろ雲 | すりガラスのように太陽がぼんやり透ける。 | 時にシトシトと連続的な雨を降らせる。 |
| **下層雲**<br/>（地上付近〜約6,500ft） | **層積雲 (Sc)** | むら雲 | 大きな黒っぽい塊が敷き詰められたような雲。 | 弱い雨や雪を伴う。 |
| | **層雲 (St)** | きり雲 | 最も低い、霧のような雲。地表や山肌にへばりつく。 | 霧雨（小雨）を降らせる。**視程障害**の元凶。 |
| | **乱層雲 (Ns)** | 雨雲 | 空を暗く覆い、本格的な雨や雪を降らせる。**層雲系**の代表格。 | **広範囲にわたる長時間の連続的な雨**。 |
| **垂直に発達する雲**<br/>（下層から上空をぶち抜く） | **積雲 (Cu)** | わた雲 / 入道雲の子供 | 晴れた日にポコポコ浮かぶ。上面がモコモコ。 | 発達すると**にわか雨（しゅう雨）**を降らせる。 |
| | **積乱雲 (CB)** | 入道雲 / カミナリ雲 | 山のようにそびえ立つ、成層圏まで届く巨大な雲。 | **激しい突風（ガスト）、雷、ヒョウ、しゅう雨**。 |

### 4. 最恐のコンボ「Embedded CB（隠れ積乱雲）」

さっきのベテランパイロットが冷や汗をかいた、最悪の地雷たい。
温帯低気圧や温暖前線の付近などで、空一面がベタッとした層状の雨雲である**「乱層雲（Ns）」**に暗く覆われとる時。
「まあ、ベタッとした雲やから、揺れは少ないやろ。突っ切るばい」と油断して入った瞬間……。

その穏やかな雨雲の中に、猛烈な上昇気流とカミナリを持つ**積乱雲（CB）がすっぽりと覆い隠されとる（Embedded：埋没しとる）**ことがあるんよ！

目で見えんけん、パイロットは気象レーダーを駆使して「この乱層雲の中に、真っ赤（ヤバい）なエコーの塊が隠れとらんか？」を常に探し出さなアカン。
見た目の大人しさに騙されたら、一瞬で地雷（隠れCB）を踏み抜くことになるんよ。

## 💡 試験対策：ここが出るばい！（Check Six）

試験官は、10種雲形の「名前の引っかけ」と「降水・対流の特徴」を組み合わせてくるぞ。

### 敵機パターン1：高積雲の層分類（名前の引っかけ）
> **問：** 雲の分類とその特徴に関する記述として、**誤っているもの**はどれか。
> 1. 高層雲は中層雲に分類され、太陽や月がすりガラスを通したようにぼんやりと見える特徴がある。
> 2. 積乱雲は下層から上層まで垂直に発達し、雷や突風、ヒョウなどを伴う激しい気象をもたらす。
> 3. 高積雲は上層雲に分類され、主に氷の結晶で構成されているため、降水をもたらすことはほとんどない。
> 4. 乱層雲は、広範囲にわたって連続的な雨や雪を降らせる厚い層状の雲である。

**答え：** **3**
（解説：出たな、高がつくのに一番上じゃないヤツ！「高積雲（ひつじ雲）」は名前が紛らわしいが**中層雲**たい！一番上の上層雲は「巻（けん）」がつくやつ（巻雲、巻層雲など）やけんね。この名前のトラップに引っかかったら、空で地雷を踏むばい！）

### 敵機パターン2：積雲系 vs 層雲系の性質
> **問：** 積雲（または積乱雲）と、乱層雲（または層雲）の気象的特徴の違いについて、**正しいもの**はどれか。
> 1. 積雲系の雲は、安定した大気中で発生し、シトシトと広範囲にわたる長時間の連続的な雨を降らせる。
> 2. 層雲系の雲は、不安定な大気中で発生し、突風や雷を伴うにわか雨（しゅう雨）を降らせる。
> 3. 積雲系の雲は、強い上昇気流（対流）によって垂直に発達し、内部では激しい乱気流（タービュランス）が発生する。
> 4. 層雲系の雲は、内部の対流活動が極めて活発なため、いかなる高度でもアイシング（着氷）が発生しない。

**答え：** **3**
（解説：雲のキャラクターの仕分けたい！ 1は「層雲系」、2は「積雲系」の特徴やから逆たい。4は「対流がない（安定）」からこそ、冬に過冷却水滴が逃げずに残り、恐ろしいアイシングの温床になるんよ。正解は3！モクモクした積雲系は、とにかくヤンチャで激しく揺れる（上昇気流）と覚えんしゃい！）

### 敵機パターン3：冬の層雲とアイシング
> **問：** 冬期の飛行において、氷結高度（Freezing Level）以下、またはそれに近い層雲内を長時間にわたって有視界（または計器）飛行する場合、最も警戒すべき危険はどれか。
> 1. 急激な上昇気流による、機体の構造破壊。
> 2. 機体表面（ピトー管やインテイク）への氷雪の付着による、急激な性能低下（アイシング）。
> 3. 視程が一時的に悪化するが、高度を一定に保てば性能への影響は全くない。
> 4. 静電気が急激に帯電することによる、落雷の発生。

**答え：** **2**
（解説：冬の東北の層雲を舐めたらいかん！層雲は大人しそうに見えるが、冬場はコップの中の冷たい水滴（過冷却水滴）がギッシリ詰まった最悪の氷結地帯たい。ここをダラダラ飛び続けると、インテイクや翼に氷（2）がみるみる育ち、エンジンを壊す。見つけたら即、高度変更して逃げるのがプロたいね！）

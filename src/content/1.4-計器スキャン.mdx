# 1.4 計器クロスチェック（計器スキャン）

計器飛行において最も重要なスキルの一つが、**「計器クロスチェック」**（または**「計器スキャン」**）です。これは、複数の計器を素早く、かつ系統的に確認し、航空機の状態を総合的に把握するための技術です。外部視界に頼れない状況では、この計器スキャンが適切に行われるかどうかが飛行の安全に直結します。

## 1.4.1 効率的なスキャンの基本

効率的な計器スキャンには、以下の基本原則があります：

1. **核となる計器に焦点を当てる：**
   姿勢指示器（ADI）を中心に据えたスキャンパターンを確立します。ADIは、航空機の姿勢に関する最も直接的な情報を提供するため、他の計器を確認する間も頻繁に戻ってくる「ホーム」の位置として機能します。

2. **関連計器をグループ化する：**
   特定の飛行フェーズや操作に関連する計器をグループとして確認します。例えば、水平直線飛行を維持する場合は、姿勢指示器、高度計、昇降計、対気速度計のグループを中心にスキャンします。

3. **適切な滞留時間：**
   各計器に適切な時間だけ視線を留めます。計器を凝視しすぎると他の計器の確認が遅れ、逆に短すぎると情報を正確に読み取れません。一般的には、各計器に約1〜2秒程度の滞留時間が適切とされています。

4. **クロスチェックの頻度を適切に保つ：**
   計器の確認は継続的に行いますが、操縦操作の複雑さや飛行フェーズに応じて、スキャンの頻度や重点を置く計器を調整します。

5. **優先順位をつける：**
   常にすべての計器を均等に確認するのではなく、その時点での飛行状況や操作に最も関連する計器に優先順位をつけます。

## 1.4.2 主要なスキャンパターン

計器スキャンには、いくつかの基本的なパターンがあります。パイロットは自分に合ったパターンを見つけ、それを訓練によって習熟させることが重要です。

### 1.4.2.1 ハブアンドスポークパターン

<Diagram 
  src="/images/hub-spoke-scan.png" 
  alt="ハブアンドスポークスキャンパターン" 
  caption="姿勢指示器を中心としたハブアンドスポークスキャンパターン" 
  height="300px"
/>

このパターンでは、**姿勢指示器（ADI）を「ハブ（中心）」として**、そこから放射状に他の計器（「スポーク」）へと視線を移動させます。一つの計器を確認した後は必ず姿勢指示器に戻り、次の計器へと移動します。例えば：

1. 姿勢指示器 → 高度計 → 姿勢指示器
2. 姿勢指示器 → 対気速度計 → 姿勢指示器
3. 姿勢指示器 → 方位指示器/HSI → 姿勢指示器
4. 姿勢指示器 → 昇降計 → 姿勢指示器
5. 姿勢指示器 → エンジン計器 → 姿勢指示器

このパターンの長所は、姿勢指示器を頻繁に確認することで航空機の基本姿勢を常に把握できる点です。特に初学者や、不慣れな状況では効果的です。

### 1.4.2.2 サーキュラーパターン

<Diagram 
  src="/images/circular-scan.png" 
  alt="サーキュラースキャンパターン" 
  caption="計器パネルを時計回りまたは反時計回りに確認するサーキュラースキャンパターン" 
  height="300px"
/>

このパターンでは、**計器パネル上の計器を時計回り（または反時計回り）に順番に確認**していきます。例えば：

姿勢指示器 → 対気速度計 → 高度計 → 昇降計 → 方位指示器/HSI → エンジン計器 → 姿勢指示器（そして繰り返し）

このパターンの長所は、すべての計器を漏れなく確認できる点です。しかし、必ずしも現在の飛行フェーズに最も関連する計器を優先的に確認できるわけではないため、状況に応じた柔軟性が必要です。

## 1.4.3 状況に応じたスキャン

異なる飛行フェーズや操作によって、重点的に確認すべき計器が変わります。ここでは、いくつかの典型的な飛行フェーズにおけるスキャンの重点を示します：

### 1.4.3.1 水平直線飛行の維持

<CalloutBox type="info" title="重点的に確認する計器">
1. 姿勢指示器（水平姿勢の維持）
2. 高度計（目標高度の維持）
3. 昇降計（垂直速度ゼロの確認）
4. 対気速度計（目標速度の維持）
5. 方位指示器/HSI（目標方位の維持）
</CalloutBox>

### 1.4.3.2 一定レートでの上昇または降下

<CalloutBox type="info" title="重点的に確認する計器">
1. 姿勢指示器（適切なピッチ姿勢の維持）
2. 昇降計（目標上昇/降下レートの維持）
3. 対気速度計（適切な速度の維持）
4. 方位指示器/HSI（目標方位の維持）
5. 高度計（現在高度と目標高度までの距離の確認）
</CalloutBox>

### 1.4.3.3 旋回

<CalloutBox type="info" title="重点的に確認する計器">
1. 姿勢指示器（適切なバンク角の維持）
2. 方位指示器/HSI（旋回の進行状況と目標方位の確認）
3. 高度計（高度の維持）
4. 対気速度計（速度の維持）
5. 昇降計（垂直速度の確認）
</CalloutBox>

### 1.4.3.4 計器進入

<CalloutBox type="info" title="重点的に確認する計器">
1. 姿勢指示器（適切な姿勢の維持）
2. 方位指示器/HSI（コース偏位と進入経路の確認）
3. グライドスロープ指示（垂直経路の確認）
4. 高度計（現在高度と決心高度の確認）
5. 対気速度計（進入速度の維持）
6. 昇降計（適切な降下レートの確認）
</CalloutBox>

## 1.4.4 計器スキャンの習得と上達

計器スキャンの習得には継続的な訓練と実践が必要です。以下に、スキャン技術を向上させるためのポイントをいくつか示します：

1. **基本からの積み上げ：**
   まずは基本的な飛行状態（水平直線飛行など）でのスキャンパターンを確立し、徐々に複雑な操作へと進みます。

2. **シミュレーターの活用：**
   フライトシミュレーターを使用して、様々な状況や異常事態でのスキャン技術を安全に練習します。

3. **認知的な負荷の管理：**
   計器からの情報を単に「見る」だけでなく、その意味を理解し、必要な操作を判断する能力を養います。

4. **継続的な評価と改善：**
   自分のスキャンパターンの効率性を定期的に評価し、弱点を特定して改善します。

5. **自動化に頼りすぎない：**
   現代の航空機には様々な自動化システムがありますが、基本的な計器飛行スキルを維持することが重要です。

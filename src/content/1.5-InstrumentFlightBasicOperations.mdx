# 1.5 計器飛行の基本操作

計器飛行では、外部の視覚的な参照なしに航空機を正確に操縦するために、特定の基本操作が重要になります。これらの基本操作は、計器飛行のあらゆる局面の基礎となるものです。

## 1.5.1 水平直線飛行の維持

水平直線飛行は、計器飛行の最も基本的な技術です。この状態を維持するためには、以下の要素を安定させる必要があります：

1. **高度の維持**
2. **方位の維持**
3. **対気速度の維持**
4. **姿勢（ピッチおよびバンク）の維持**

  height="300px"
/>

### 1.5.1.1 操作手順

**1. 適切な姿勢の確立：**
   - 姿勢指示器（ADI）を基準として、水平姿勢（ピッチ0度、バンク0度）を確立します。
   - 巡航速度に応じた適切なピッチ姿勢を維持します（通常は地平線に対してわずかに機首上げ）。

**2. パワー設定：**
   - 目標速度を維持するために適切なパワー設定を行います。
   - エンジン計器（RPM、マニフォールド圧、N1など）を確認し、適切な値に調整します。

**3. トリムの調整：**
   - 操縦桿にかかる圧力を軽減するために、ピッチトリムを適切に調整します。
   - これにより、パイロットは余計な力を入れることなく安定した姿勢を維持できます。

**4. 継続的な監視と修正：**
   - 姿勢指示器、高度計、昇降計、対気速度計、方位指示器を継続的にスキャンします。
   - 偏差が生じた場合は、小さな修正を行い、オーバーコントロールを避けます。

<CalloutBox type="warning" title="注意点">
水平直線飛行では、常に小さな修正を心がけ、大きな修正は避けてください。大きな修正は航空機の安定性を損ない、新たな問題を引き起こす可能性があります。
</CalloutBox>

## 1.5.2 上昇と降下

上昇と降下は、計器飛行における基本的な操縦操作です。これらの操作は、高度変更や進入、出発などの場面で頻繁に行われます。

### 1.5.2.1 一定レートでの上昇

<Diagram 
  src="/images/climb-instruments.png" 
  alt="上昇時の計器表示" 
  caption="一定レートでの上昇時の主要計器の表示例" 
  height="300px"
/>

**操作手順：**

1. **上昇姿勢の確立：**
   - 姿勢指示器を参照し、適切な上昇ピッチ姿勢を確立します（通常は約5〜10度の機首上げ）。
   - バンク角は0度（水平）を維持します。

2. **パワーの増加：**
   - 上昇に必要なパワーを適用します（多くの場合、最大連続出力または指定された上昇出力）。
   - エンジン計器を確認し、制限内であることを確認します。

3. **速度の安定：**
   - 対気速度計を監視し、目標上昇速度（通常はVy：最良上昇率速度、またはVx：最良上昇角速度）を維持します。
   - 必要に応じてピッチ姿勢を微調整します。

4. **上昇レートの確認と調整：**
   - 昇降計を確認し、目標上昇レート（例：500 ft/minや1000 ft/min）が達成されていることを確認します。
   - 上昇レートを調整するには、主にピッチ姿勢を変更します。

5. **方位の維持：**
   - 方位指示器またはHSIを監視し、指定された方位を維持します。
   - 必要に応じて小さなバンク修正を行います。

6. **接近監視：**
   - 高度計を監視し、目標高度への接近を確認します。
   - 目標高度の約10%の上昇レート（例：1000 ftの上昇なら100 ft/min）手前で水平飛行への移行を開始します。

### 1.5.2.2 一定レートでの降下

<Diagram 
  src="/images/descent-instruments.png" 
  alt="降下時の計器表示" 
  caption="一定レートでの降下時の主要計器の表示例" 
  height="300px"
/>

**操作手順：**

1. **降下姿勢の確立：**
   - 姿勢指示器を参照し、適切な降下ピッチ姿勢を確立します（通常は約5度の機首下げ）。
   - バンク角は0度（水平）を維持します。

2. **パワーの減少：**
   - 降下に適したパワー設定に減少させます。
   - 目標降下率と速度に応じてパワーを調整します。

3. **速度の安定：**
   - 対気速度計を監視し、目標降下速度を維持します。
   - 必要に応じてピッチ姿勢とパワーを調整します。

4. **降下レートの確認と調整：**
   - 昇降計を確認し、目標降下レート（例：500 ft/minや1000 ft/min）が達成されていることを確認します。
   - 降下レートを調整するには、主にピッチ姿勢とパワーの組み合わせを使用します。

5. **方位の維持：**
   - 方位指示器またはHSIを監視し、指定された方位を維持します。
   - 必要に応じて小さなバンク修正を行います。

6. **接近監視：**
   - 高度計を監視し、目標高度への接近を確認します。
   - 目標高度の約10%の降下レート手前で水平飛行への移行を開始します。

<CalloutBox type="info" title="パワーとピッチの関係">
降下操作では、一般的に「パワー」が速度を、「ピッチ」が降下率を主に制御します。ただし、これらは互いに影響し合うため、両方の要素を適切に調整する必要があります。
</CalloutBox>

## 1.5.3 標準レート旋回

標準レート旋回（Standard Rate Turn）は、2分間で360度の旋回を完了する旋回率です。これは毎分3度の旋回率に相当し、多くの計器飛行手順で指定される標準的な旋回率です。

<Diagram 
  src="/images/standard-rate-turn.png" 
  alt="標準レート旋回の計器表示" 
  caption="標準レート旋回時の主要計器の表示例" 
  height="300px"
/>

### 1.5.3.1 操作手順

1. **開始姿勢の確立：**
   - 姿勢指示器を参照し、適切なバンク角を確立します。
   - 標準レート旋回のバンク角は、対気速度によって異なりますが、一般的な目安は以下の通りです：
     - 100 knots: 約15度
     - 150 knots: 約20度
     - 200 knots: 約25〜30度
   - より正確には「バンク角 = 対気速度(knots) ÷ 10 + 7」という公式が使用できます。

2. **旋回率の確認：**
   - 旋回指示器（Turn Coordinator）または旋回率計（Turn Rate Indicator）を確認し、指針が標準レートマーク（通常は「2」と表示）に一致していることを確認します。
   - 必要に応じてバンク角を微調整します。

3. **高度の維持：**
   - 旋回中は高度を維持するために、わずかに機首を上げる（ピッチを増加させる）必要があることが多いです。
   - 高度計と昇降計を監視し、必要に応じてピッチ姿勢を調整します。

4. **対気速度の維持：**
   - 対気速度計を監視し、目標速度を維持します。
   - 旋回中は揚力の一部が水平方向に向けられるため、速度が低下する傾向があります。必要に応じてパワーを増加させます。

5. **目標方位の接近監視：**
   - 方位指示器またはHSIを監視し、目標方位への接近を確認します。
   - 目標方位の約半分の旋回率（標準レート旋回の場合は約15度）手前で旋回を終了する準備を始めます。

6. **旋回の終了：**
   - 目標方位に接近したら、姿勢指示器を参照しながらロールアウト（水平状態への復帰）を開始します。
   - ロールアウトは、バンク角と同じ度数だけ目標方位の手前から始めると、ちょうど目標方位で完了します（例：20度のバンクなら、目標方位の20度手前からロールアウト開始）。

<CalloutBox type="warning" title="旋回中の注意点">
旋回中は空間識失調のリスクが高まります。計器への依存度を高め、身体感覚に頼らないようにしてください。特に急なバンク角での旋回や、長時間の旋回後は注意が必要です。
</CalloutBox>

## 1.5.4 異常姿勢からの回復

異常姿勢（Unusual Attitude）とは、航空機が意図しない危険な姿勢になった状態を指します。これは、気象条件、パイロットの注意散漫、空間識失調などによって発生する可能性があります。計器飛行では、異常姿勢からの回復手順を理解し練習することが非常に重要です。

### 1.5.4.1 異常姿勢の識別

異常姿勢は、以下の計器指示から識別できます：

**急激な機首上げ（Nose-High Attitude）:**
- 姿勢指示器に大きな機首上げ姿勢が表示される
- 対気速度が急速に減少している
- 高度が増加している、または増加率が異常に高い
- 昇降計が急上昇を示している

**急激な機首下げ（Nose-Low Attitude）:**
- 姿勢指示器に大きな機首下げ姿勢が表示される
- 対気速度が急速に増加している
- 高度が減少している、または減少率が異常に高い
- 昇降計が急降下を示している

### 1.5.4.2 回復手順

**急激な機首上げからの回復:**

<Diagram 
  src="/images/nose-high-recovery.png" 
  alt="機首上げ姿勢からの回復" 
  caption="急激な機首上げ姿勢からの回復手順" 
  height="250px"
/>

1. **パワーの増加：**
   - 最大パワーを適用して失速を防止します。

2. **バンクの導入：**
   - 軽度から中程度のバンク（約15〜20度）を導入します。
   - これにより、揚力ベクトルが水平方向に傾き、機首下げに寄与します。

3. **機首下げ：**
   - 姿勢指示器を参照しながら、前方に操縦桿を押して機首を下げます。
   - 過度の負のGを避けるため、制御された方法で行います。

4. **水平飛行への復帰：**
   - 水平線が姿勢指示器の中央に近づいたら、バンクを解除します。
   - パワーを調整し、水平直線飛行を確立します。

**急激な機首下げからの回復:**

<Diagram 
  src="/images/nose-low-recovery.png" 
  alt="機首下げ姿勢からの回復" 
  caption="急激な機首下げ姿勢からの回復手順" 
  height="250px"
/>

1. **パワーの減少：**
   - アイドルまたは適切な減少位置にパワーを減らし、速度の増加を抑制します。

2. **過度のバンクの修正：**
   - もし存在する場合、過度のバンクを修正して水平に戻します。
   - 機首上げ操作を行う前に、翼を水平に近づけることが重要です。

3. **機首上げ：**
   - 姿勢指示器を参照しながら、後方に操縦桿を引いて機首を上げます。
   - 過度の正のGを避けるため、制御された方法で行います。

4. **水平飛行への復帰：**
   - 水平線が姿勢指示器の中央に近づいたら、パワーを調整します。
   - 水平直線飛行を確立し、速度と高度を安定させます。

<CalloutBox type="danger" title="重要な注意点">
異常姿勢からの回復中は、過度の操作や急激な操作を避け、航空機に過大な応力をかけないようにしてください。特に高速状態での急激な引き起こしや、低速状態での急激なバンクは避けるべきです。常に航空機の運用限界内で操作してください。
</CalloutBox>

### 1.5.4.3 訓練の重要性

異常姿勢からの回復は、定期的な訓練と練習が必要なスキルです。実際の緊急事態では、短時間で正確な判断と操作が求められます。フライトシミュレーターや安全な高度での実地訓練を通じて、これらの手順に習熟することが重要です。

## 1.5.5 運用上の考慮事項

計器飛行の基本操作を効果的に実行するために、以下の運用上の考慮事項に留意してください：

### 1.5.5.1 ワークロード管理

1. **優先順位付け：**
   - 「Aviate, Navigate, Communicate」（飛行、航法、通信）の優先順位を維持します。
   - 常に航空機の制御を最優先し、その後に航法、そして最後に通信を行います。

2. **タスクの分散：**
   - 複雑な操作は、可能な限り小さなタスクに分解します。
   - 高ワークロード環境では、不必要なタスクを延期または委任します。

### 1.5.5.2 CRM（Crew Resource Management）

1. **コミュニケーション：**
   - 複数のクルーがいる場合、明確なコミュニケーションを維持します。
   - 意図と行動を明確に伝え、誤解を防止します。

2. **役割分担：**
   - 操縦担当パイロット（PF）と監視担当パイロット（PM）の役割を明確に分担します。
   - PMは計器のクロスチェックと偏差の通知を担当します。

### 1.5.5.3 自動化の活用と監視

1. **適切な自動化レベル：**
   - 状況に応じて適切な自動化レベルを選択します。
   - 高ワークロード環境では自動化を活用し、ワークロードを軽減します。

2. **継続的な監視：**
   - 自動化システムの動作を継続的に監視します。
   - 「自動化の落とし穴」を避けるため、状況認識を維持します。

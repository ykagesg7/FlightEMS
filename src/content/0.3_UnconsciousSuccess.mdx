# 「なんか知らんけど上手くいった」…それ、ホントに偶然？

こんにちは！しゃどーです。

最近、「できた！」っていう瞬間、ありましたか？こういう瞬間って、いくつになってもいいですよね。

でも、そういう時って「**たまたま運が良かっただけかも**」とか、「**次も上手くいくか分からないな…**」なんて思っちゃったりしません？

世の中には、「成功は再現性が大事！」みたいな話もよく聞きますよね。確かにそれも一理ある。でも、毎回同じようにやって成功するほど、世の中単純じゃない気もするんですよね。

**「じゃあ、どうすりゃいいのさ？」** って思いますよね。わかります。僕も昔はそうでした。

いろいろ試行錯誤してきた中で、最近すごく思うのは、<Highlight> 「**自分で考えて、実際にやってみる**」</Highlight> ことがめちゃくちゃ大事なんじゃないか、ってことです。

有名な『7つの習慣』って本、読んだことありますか？あの本にも「主体的であること」が最初の習慣として出てくるんですよね。

これ、要するに<Highlight>「**周りの状況や他人のせいにしないで、まず自分で考えて行動しようぜ！**」</Highlight>ってことだと僕は解釈してます。**考えるだけで終わらない**、ここがポイントな気がします。

僕自身の経験で言うと、昔は「まあ、誰かがやってくれるっしょ」みたいに思ってた部分もあったんですが（笑）、『7つの習慣』を読んで、「いや、自分で動かないと何も始まらんわ」と考えを改めたんです。そして、**実際に小さなことから行動に移す**ようにしました。

そしたら不思議なことに、**「これもやってみない？」** って声をかけてもらえる機会が明らかに増えたんですよね。もちろん、全部が全部上手くいくわけじゃないですよ。失敗もたくさんしました。

でも、自分で「これをやる！」って決めて**行動**した経験って、成功しようが失敗しようが、なんか<Highlight>**自分の中に具体的な経験値として貯まっていく感じ**</Highlight>がするんです。頭で考えてるだけじゃ、この経験値は貯まらないんですよね。

だから、「できた！」って経験をしたあなたに伝えたいのは、その成功体験は「**あなたが主体的に考え、そして実際に行動したからこそ得られたもの**」 なんじゃないか？ってことです。

もちろん、周りのサポートとか、タイミングとか、いろんな要素が絡み合っての結果だとは思います。でも、その**きっかけとなる最初の一歩を踏み出した**のは、間違いなくあなた自身の「やってみよう」っていう**行動**だったはず。

とはいえ、「主体的に行動しろって言われても、何をすれば…？」ってなる人もいるかもですね。

そんなに難しく考えなくていいと思うんです。

*   **まずは、自分の「気になる」「やってみたい」という考えを、具体的な行動計画に落とし込んでみる。**
*   **小さなことでいいから、「自分で決めて、実際にやってみる」経験を増やす。（例：気になった本をまず1ページ読んでみる、興味のあるイベントに申し込んでみる、など）**
*   **上手くいかなくても、「次はこうしてみよう」と考え、また行動してみる。**

**考えるサイクルで止めずに、行動するサイクルを回す**こと。これだけでも、だいぶ「自分で舵を取ってる感」が出てくるんじゃないかな。

というわけで、

「なんか知らんけど上手くいった」と感じたその成功体験。それはきっと、あなたが無意識のうちに「主体的」に考え、そして**実際に行動した**結果なのかもしれません。その**行動力**を大事にして、これからもどんどん自分で考えて、**具体的なアクション**を起こしてみてほしいなと思います。

そしたら、きっともっと面白い経験がたくさんできるはずですよ！

# CPL航空法 Mission 0: 完全攻略ブリーフィング

## 🎯 このミッションの目標

CPL（事業用操縦士）試験の航空法規分野を完全攻略するための基礎知識を身につけます。

## 📋 学習内容概要

### 1. CPL航空法の全体像
- 航空法の体系と構造
- CPL試験での出題傾向
- 効率的な学習方法

### 2. 重要法規の優先順位
1. **技能証明制度** (最重要)
2. **航空身体検査証明** (重要)
3. **航空機の耐空性** (重要)
4. **飛行規則・管制** (標準)
5. **事故調査・責任** (標準)

## 🎓 学習戦略

### 効率的な攻略法
1. **頻出問題から攻略**
   - 過去5年間の出題分析
   - 重要度別学習計画

2. **体系的理解**
   - 法規の関連性を理解
   - 実務との関連付け

3. **反復練習**
   - 問題演習での定着
   - 弱点分野の重点学習

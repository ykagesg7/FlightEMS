## 5. デジタルノータムリクエストサービス

デジタルノータムリクエストサービスの操作について説明する。

デジタルノータムリクエストサービスは、デジタルノータムの検索を実施する機能である。「**デジタルノータム検索**」画面より検索条件を入力してデジタルノータムを検索し、ポリゴン情報を持つデジタルノータムについては地図上に表示する。

### 5.1 デジタルノータムの検索

ブラウザからのデジタルノータムの検索リクエストに対して、データベースからデジタルノータムを取得し、検索結果の表示を行う。

#### （1）操作フロー

デジタルノータムの検索は、以下の手順で行う。

```text
（デジタルノータムの検索 開始）
       ↓
【利用者端末】
<1> 「デジタルノータム検索」画面の表示
       ↓
<2> 検索条件の入力
       ↓
<3> デジタルノータムの検索
       ↓
（デジタルノータムの検索 終了）
```

> **図 5.1-1** デジタルノータムの検索（1）（原本 PDF 参照）

#### （2）操作手順

1. SWIM ポータルの「利用サービス一覧」画面からデジタルノータムリクエストサービスのカードを押下し、「デジタルノータム検索」画面を表示する。
2. 「デジタルノータム検索」画面の「検索条件」を入力する。検索条件の詳細は **表 5.1-1** に示す。
3. **[検索]** ボタンを押下する。

> **図 5.1-2** デジタルノータムの検索（2）  
> **図 5.1-3** デジタルノータムの検索（テーブル形式）  
> **図 5.1-4** デジタルノータムの検索（リスト形式）

検索結果が表示される。（表示形式は「テーブル形式」または「リスト形式」を選択可能）

**表 5.1-1 手順 \<2\> の検索条件**

| 名称 | 概要 |
|------|------|
| 「NOF」 | NOF を入力する。（完全一致検索） |
| 「FIR」 | FIR を入力する。（完全一致検索） |
| 「ロケーション」 | ロケーションを入力する。（半角スペースで区切られた個々の単語を完全一致検索） |
| [ロケーション] コンボボックス | ロケーションを選択する。 |
| 「ノータムコード」 | ノータムコードを入力する。（前方一致検索） |
| 「シリーズ」 | シリーズを入力する。（完全一致検索） |
| 「ノータム番号」 | ノータム番号を入力する。（完全一致検索） |
| 「UUID」 | UUID を入力する。（前方一致検索） |
| [範囲] チェックボックス | 範囲を選択する。（飛行場／エンルート／ワーニング） |
| 「下限／上限」 | 下限／上限を入力する。 |
| 「キーワード」 | キーワードを入力する。（半角スペースで区切られた個々の単語を完全一致検索） |
| [AND/OR] ラジオボタン | AND／OR を選択する。 |
| [有効/無効] ラジオボタン | 有効／無効を選択する。 |
| 「有効日時」 | 有効日時を入力する。 |
| [表示件数] ラジオボタン | 検索結果の表示件数を選択する。（25／50／100） |

### 5.2 デジタルノータムの地図表示

ブラウザからのデジタルノータムの検索リクエストに対して、データベースからデジタルノータムを取得し、地図の表示を行う。

#### （1）操作フロー

デジタルノータムの地図表示は、以下の手順で行う。

```text
（デジタルノータムの地図表示 開始）
       ↓
【利用者端末】
<1> 「デジタルノータム検索」画面の表示
       ↓
<2> 検索条件の入力
       ↓
<3> デジタルノータムの地図表示
       ↓
（デジタルノータムの地図表示 終了）
```

> **図 5.2-1** デジタルノータムの地図表示（1）

#### （2）操作手順

1. SWIM ポータルの「利用サービス一覧」画面からデジタルノータムリクエストサービスのカードを押下し、「デジタルノータム検索」画面を表示する。
2. 「デジタルノータム検索」画面の「検索条件」を入力し、**[検索]** ボタンを押下する。「検索条件」の詳細は **表 5.1-1** に示す。

> **図 5.2-2** デジタルノータムの地図表示（2）

3. 検索結果から、ノータムを選択する。

> **図 5.2-3** デジタルノータムの地図表示（3）

**【補足】** 地図上に表示できないノータムの図形については、地図表示領域中央部に「グラフィックなし（No image）」が表示される。

### 5.3 デジタルノータムの GML ダウンロード

デジタルノータムの検索を行い、検索結果のデジタルノータムの GML ダウンロードを行う。

#### （1）操作フロー

デジタルノータムの GML ダウンロードは、以下の手順で行う。

```text
（デジタルノータムの GML ダウンロード 開始）
       ↓
【利用者端末】
<1> 「デジタルノータム検索」画面の表示
       ↓
<2> 検索条件の入力
       ↓
<3>、<4> デジタルノータムの GML ダウンロード
       ↓
（デジタルノータムの GML ダウンロード 終了）
```

> **図 5.3-1** デジタルノータムの GML ダウンロード（1）

#### （2）操作手順

1. SWIM ポータルの「利用サービス一覧」画面からデジタルノータムリクエストサービスのカードを押下し、「デジタルノータム検索」画面を表示する。
2. 「デジタルノータム検索」画面の「検索条件」を入力し、**[検索]** ボタンを押下する。「検索条件」の詳細は **表 5.1-1** に示す。

> **図 5.3-2** デジタルノータムの GML ダウンロード（2）

3. 検索結果から、GML ダウンロードを行うノータムを選択する。
4. **[GML ダウンロード]** ボタンを押下する。

> **図 5.3-3** デジタルノータムの GML ダウンロード（3）  
> **図 5.3-4** デジタルノータムの GML ダウンロード（4）

選択したノータムの GML がダウンロードされる。

### 5.4 デジタルノータムの XML ダウンロード

デジタルノータムの検索を行い、検索結果のデジタルノータムの XML ダウンロードを行う。

#### （1）操作フロー

デジタルノータムの XML ダウンロードは、以下の手順で行う。

```text
（デジタルノータムの XML ダウンロード 開始）
       ↓
【利用者端末】
<1> 「デジタルノータム検索」画面の表示
       ↓
<2> 検索条件の入力
       ↓
<3> デジタルノータムの XML ダウンロード
       ↓
（デジタルノータムの XML ダウンロード 終了）
```

> **図 5.4-1** デジタルノータムの XML ダウンロード（1）

#### （2）操作手順

1. SWIM ポータルの「利用サービス一覧」画面からデジタルノータムリクエストサービスのカードを押下し、「デジタルノータム検索」画面を表示する。
2. 「デジタルノータム検索」画面の「検索条件」を入力し、**[ダウンロード]** ボタンを押下する。「検索条件」の詳細は **表 5.1-1** に示す。  
   ※検索結果を選択するのではなく、**検索条件に合致する全件を XML で一括ダウンロード**する機能である。

> **図 5.4-2** デジタルノータムの XML ダウンロード（2）

3. ダウンロードされた zip ファイルをダブルクリックする。

> **図 5.4-3** デジタルノータムの XML ダウンロード（3）  
> **図 5.4-4** デジタルノータムの XML ダウンロード（4）

XML ファイルが表示される。

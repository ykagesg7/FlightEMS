## 1. 情報サービスの名称

デジタルノータムリクエストサービス

## 2. 情報サービスのバージョン

1.0.0

## 3. 情報サービスのライフサイクル・ステータス

SWIM portal のサービスライフサイクルステータスを参照

## 4. 情報サービスのライフサイクル日

2026.3.4（日本時間）

## 5. 情報サービスの機能

航空局のデータベースから、検索条件に応じたノータムを **Request/Reply** により確認できる。

## 6. 情報の分類

航空情報

## 7. 情報サービスの概要

ノータムを取得するためのサービス。

## 8. 情報サービスについての追加情報

NIL

## 9. サービスの品質

### 性能に関するパラメータ

処理の **95%** に対して、要求受信又は配信トリガの発生から **1 秒以内**に配信開始。

### 信頼性に関するパラメータ（定量的）

夜間保守時を除き、**24 時間 365 日**サービス無停止。サーバ設置官署被災時のバックアップ機能あり。

### セキュリティパラメータ（定性的）

ID、パスワードによるログイン認証を行う。

## 10. 情報サービスの検証形式

自己検証

## 11. 情報サービスの検証の説明

航空局による自己検証を実施

## 12. 利用可能なフィルタリング

NIL

## 13. アクセス制限

| 区分 | 内容 |
|------|------|
| 利用者 | SWIM portal アカウントを有する者 |
| システム連携 | Web API |

Web API を利用する場合は、利用に際し情報サービス提供者に**届出を提出**する。詳細な利用手続きは**サービス説明書**で確認できる。

## 14. メッセージ交換パターン

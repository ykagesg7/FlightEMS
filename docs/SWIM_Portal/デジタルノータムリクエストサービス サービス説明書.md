## 1. 目的

このサービス説明書は、SWIM 運用規程に基づき、デジタルノータムリクエストサービスにおける利用手続や技術情報について説明するものである。

## 4. サービス利用手続き

### (1) 情報サービス利用者の対象及び追加書類

- **対象:** SWIM portal のアカウントを有する者  
- **追加書類:** サービス利用申請書（**Web API を利用する場合に限る**）

### (2) サービス利用手続き及び期間

- Web API を利用する場合におけるサービス利用手続きに係る流れは、「**情報サービス利用手続きフローチャート**」（本書末尾）のとおり。
- サービス利用申請後、すみやかに (1) の追加書類を提出すること（Web API を利用する場合に限る）。
- Web API を利用する場合、サービス利用申請から審査結果通知までの期間は、**標準的に 2 週間程度**要し、調整の状況によって期間が変動する場合がある。

### (3) システム連携の留意事項

- 接続回線は**インターネット回線**とし、セキュリティ要件は必要としない。
- 接続回線の敷設及び維持に係る費用は**申請者負担**とする。
- 通信方式は、技術資料（API 連携仕様書等）に適合すること。

## 5. サービスの説明

### 5.1 推奨環境

一般的な IP ネットワーク（IPv4, IPv6）を利用するインターネット接続環境であれば、サービスを利用できる。推奨ブラウザは **Google Chrome** 及び **Safari** である。

### 5.2 機能詳細

航空局のデータベースから、検索条件に応じたノータムを **Request/Reply** により確認できる。

### 5.3 システム連携

**Web API**

## 6. 留意事項

1. 日本の **SNOWTAM** は、**S/T シリーズ**で提供される。
2. 検索結果が膨大となる場合、**エラーを返す**。
3. 情報サービス提供者は、障害等により情報サービスが利用不可となった場合には、その旨を **SWIM portal** において周知する（SWIM portal が運用中の場合に限る）。なお、ノータムの確認が必要な場合、**AIP GEN 3.1** に記載の空港事務所または航空情報センターに問い合わせること。
4. **SWIM 利用規約 別紙 2「SWIM のコンテンツの利用について」2）及び 4）**に関する具体例:

#### a. 第三者に権利がある場合の具体例

【SWIM コンテンツの利用について 2）関係】

- 外国のノータムは、当該国機関等が著作権その他の権利を有している場合がある。利用する場合は、情報サービス利用者の責任において確認すること。
- 地理院タイル（表示される地図等）は、外部データベース等との API 連携等により取得している。国土地理院の以下のページを参照すること。  
  - [国土交通省国土地理院 地図・空中写真閲覧サービス（開発用ページ一覧）](https://maps.gsi.go.jp/development/ichiran.html)

#### b. 別のルールを適用するコンテンツ

【SWIM コンテンツの利用について 4）関係】

「SWIM コンテンツの利用について」の規定に関わらず、以下のとおりとする。

- 原則として本サービスで提供するコンテンツを複製、公衆送信、翻訳・変形等の翻案等に利用する場合、利用目的や利用者は問わない。
- 本サービスで提供するコンテンツを第三者が利用可能なよう公衆送信等を行う場合は以下の措置を講じること。
  - **航空機の運航に利用する場合:** 最新の正しい情報を維持する。これが保証できない場合は、最新の正しい情報ではない可能性があることを**明示**する。
  - **航空機の運航に利用しない場合（利用される可能性がある場合）:** 航空機の運航に利用できないことを**明示**する。ただし、航空機の運航に利用されないことが明らかな場合はこの限りではない。
- 本サービスが提供するコンテンツを利用する場合、**出典の明記は不要**とする。国（又は府省等）及び情報サービス提供者が作成したかのような態様で公表・利用しなければ、編集・加工等を行ったことの記載も不要とする。

### 情報サービス利用手続フローチャート（Web API 利用時）

```text
【情報サービス利用者】                      【情報サービス提供者】

1. SWIM portal アカウント作成・ログイン
        ↓
2. サービス利用申請 -----------------------> SWIM portal からの申請受信
        ↓                                        ↓
3. 追加書類（利用申請書）の提出 ---------------> （メールで受信）
   （宛先: helpdesk@ais.mlit.go.jp）             ↓
        ↓                                  4. 審査（※標準 2 週間程度）
        |                                   必要に応じた問い合わせ
        |                                        ↓
        |<--------------------------------- 5. 審査結果通知（URL 付 API 連携仕様書）
        ↓
6. API 連携仕様書を参照しログイン API を実行
        ↓
7. ログイン成功（認証情報 Cookie を保存）
        ↓
8. 認証情報を HTTP ヘッダに格納し、目的の API を実行
        ↓
9. サービス利用開始（情報入手）

（※ブラウザのみによる利用の場合、上記の手順 2 以降の API 関連手続きは不要）
```

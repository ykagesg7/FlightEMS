# SWIM Portal / デジタルノータム（参考資料）

国土交通省航空局 SWIM の **デジタルノータムリクエストサービス（S2019）** および関連 API 仕様の **社内参照用 Markdown** を置くフォルダです。原本は PDF（航空局発行）が正本です。

## 含まれるファイル

| ファイル | 内容 |
|----------|------|
| [SWIM_API_Common.md](./SWIM_API_Common.md) | SWIM サービス API 連携仕様書・**共通編**（WebAPI / AMQP、Cookie 認証、別紙サービス一覧） |
| [SWIM_API_DigitalNOTAM.md](./SWIM_API_DigitalNOTAM.md) | 付録 04・デジタルノータム**検索 API**（`f2dnrq/web/search`、JSON 内 AIXM XML） |
| [デジタルノータムリクエストサービス サービス説明書.md](./デジタルノータムリクエストサービス%20サービス説明書.md) | 利用手続、留意事項、申請フロー・利用申請書テンプレ |
| [デジタルノータムリクエストサービス ユーザーズガイド.md](./デジタルノータムリクエストサービス%20ユーザーズガイド.md) | ポータル UI 操作（検索・地図・GML/XML ダウンロード） |
| [デジタルノータムリクエストサービス 情報サービス概要.md](./デジタルノータムリクエストサービス%20情報サービス概要.md) | サービス概要・品質・連絡先 |

## PDF 原本

`PDF/` に原本 PDF を置けるが、リポジトリの `.gitignore`（`**/*.pdf`）により **Git には含めない**想定。ローカル保管または公式サイトから取得すること。

## アプリ実装（未接続）

本リポジトリのアプリは **現時点では SWIM API に未接続**。将来、NOTAM を表示する場合は:

- **サーバ側プロキシ**（認証 Cookie・秘密情報をブラウザに出さない）
- **準静的データは GeoJSON 等**、NOTAM は **オンデマンド取得＋短い TTL キャッシュ**
- サービス説明書の **免責・運航利用に関する表示** に従う

詳細は `docs/05_設計仕様書.md` の飛行計画／地図周りの記述と上記仕様書を参照。

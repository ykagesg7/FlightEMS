## 別紙2-1：WebAPIインタフェース仕様 (ログイン要求)

* **サービス名:** SWIM管理サービス
* **API名:** ログイン要求

### 1. 概要
ワンストップ機能（Web API）を利用するプログラムから呼び出され、本APIで指定されたユーザIDパスワードでユーザ認証を行う。

### 2. API詳細
#### (1) 文字コード
UTF-8

#### (2) リクエスト
**リクエスト形式**
* **URL:** `https://top.swim.mlit.go.jp/swim/webapi/login`
* **メソッド:** POST

**HTTPヘッダ**

| 項目名 | 項目名(日本語) | 設定内容 | 必須 | 備考 |
|---|---|---|:---:|---|
| `Content-Type` | 要求データの形式 | `application/json` | ◯ | |

**リクエストパラメータ（JSONボディ内）**

| 項目名(英字) | 項目名(日本語) | 設定内容 | 必須 | 備考 |
|---|---|---|:---:|---|
| `id` | アカウントID | アカウントID（メールアドレス） | ◯ | |
| `password` | パスワード | パスワード | ◯ | |

**HTTPボディサンプル**
```json
{
  "id":"test001@swim.com",
  "password":"password"
}
```

## 別紙4-1：WebAPIインタフェース仕様 (ログイン応答)

* **サービス名:** SWIM管理サービス
* **API名:** ログイン応答

### 1. 概要
ワンストップ機能（Web API）を利用するプログラムから呼び出され、本APIで指定されたユーザIDパスワードでユーザ認証を行いその結果を返却する。
認証が成功した場合は、認証済情報ヘッダを払い出しHTTPヘッダに付加する。

### 2. API詳細
#### (1) 文字コード
UTF-8

#### (2) レスポンス
**HTTPステータスコード: 200**

| HTTPボディ（エラー情報）エラーコード | HTTPボディ（エラー情報）エラー詳細 | 意味 |
|---|---|---|
| `0` | - | 正常終了 |

**HTTPヘッダ**

| 項目名 | 項目名(日本語) | 設定内容 | 備考 |
|---|---|---|---|
| `Content-Type` | 応答データの形式(MIMEタイプ) | `application/json` | |
| `Set-Cookie` | 認証済情報ヘッダ | `MSMSI=<セッション情報>; Domain={MASSのドメイン名}; secure` | |
| `Set-Cookie` | 認証済情報ヘッダ | `MSMAI=<属性情報>; Domain={MASSのドメイン名}; secure` | |

**HTTPボディ**

| 項目名(英字) | 項目名(日本語) | 設定内容 | 型 | 備考 |
|---|---|---|---|---|
| `error_info` | エラー情報 | | - | |
| `error_code` | エラーコード | | int | |
| `error_description`| エラー詳細 | | string | |

**HTTPボディサンプル**
```json
{
  "error_info": {
    "error_code": 0,
    "error_description": ""
  }
}
```

#### (3) エラーレスポンス
| HTTPステータスコード | HTTPボディ（エラー情報）エラーコード | HTTPボディ（エラー情報）エラー詳細 | 意味 |
|---|---|---|---|
| 400 | - | - | 渡されたパラメータが異なるなど、リクエストが正しくない |
| 401 | - | - | 認証失敗 |
| 404 | - | - | 要求したリソースが存在しない |
| 500 | - | - | サーバ内部でエラーが発生した |

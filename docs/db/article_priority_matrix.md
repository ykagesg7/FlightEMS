## 優先順位決定基準

### 評価軸

1. **重要度スコア** (Importance Score)
   - 8.0以上: 最高優先度
   - 7.0-7.9: 高優先度
   - 5.0-6.9: 中優先度
   - 5.0未満: 低優先度

2. **出題頻度** (Appearance Frequency)
   - 高頻出: 複数回出題（頻度 > 0.5）
   - 中頻出: 1-2回出題（頻度 0.1-0.5）
   - 低頻出: 初出題または未出題（頻度 < 0.1）

3. **問題数** (Question Count)
   - 3問以上: 高価値コンテンツ
   - 1-2問: 標準コンテンツ
   - 0問: 要追加調査

4. **科目バランス**
   - 全科目を均等にカバー
   - ユーザーの弱点科目を優先

## 優先順位変更のトリガー

以下の条件に該当する場合は、優先順位を再評価する：

1. **ユーザーフィードバック**
   - 特定トピックの要望が増加
   - 正答率が低いトピックの特定

2. **出題傾向の変化**
   - 新規問題の追加
   - 重要度スコアの更新

3. **データ分析結果**
   - ユーザーの弱点科目の特定
   - 学習効果の測定結果
